# 🏛️ مدينة+ | Baghdad Urban Intelligence Platform

> منصة الذكاء الحضري المتكامل لبغداد
> بناء: ETC GeoAI Solutions Division

## المشكلة التي تحلها
بغداد تمتلك 2,400+ مجمع سكني بلا رقابة رقمية،
شبكة طاقة تنهار بلا إنذار، وفيضانات متكررة
لأن البلدية لا ترى شبكة الصرف على خريطة.

## الحل
منصة ذكاء مكاني متكاملة تربط:
- Complex+ (إدارة المجمعات)
- Workflow+ (أتمتة المهام)  
- GIS + AI (التحليل والتنبؤ)

## التقنيات
Python | GeoPandas | Folium | Scikit-learn
PostGIS | FastAPI | Prophet | XGBoost
